# WebPageApplication

Simple Node HTTP server that serves a static `public/index.html` page.

What it is
- `index.js` — small Node HTTP server listening on port 8080 and serving `public/index.html`.
- `public/` — contains the frontend `index.html`.

Prerequisites
- Node.js

Quick start

```bash
cd WebPageApplication
npm install
npm start
# open: http://localhost:8080
```
